# 🚖 Taxi Booking System

## 📌 Project Overview

This is a web-based taxi booking system developed using PHP, MySQL, HTML, CSS, and JavaScript. It allows users to book taxis, drivers to be assigned, and rides to be completed with payment and feedback.

## ⚙️ Technologies Used

* PHP
* MySQL (XAMPP)
* HTML, CSS, JavaScript

## 📂 Features

* User Registration & Login
* Book Taxi
* Driver Assignment
* Ride Completion
* Payment Processing
* Driver Rating
* Receipt Generation

## 🗄️ Database Setup

1. Open phpMyAdmin (http://localhost/phpmyadmin)
2. Create a database named `taxi_booking`
3. Import the file from `/database/schema.sql`

## 🚀 How to Run

1. Install XAMPP
2. Start Apache & MySQL
3. Place project folder in `htdocs`
4. Open browser:
   http://localhost/TAXI_BOOKING/public/register.html

## 📁 Folder Structure

* public → frontend files
* src → backend PHP logic
* config → database connection
* database → SQL file
* docs → documentation
